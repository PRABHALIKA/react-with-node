To run this application fallow below steps

1. Etract node folder.

2. Go to node folder with command propmt and run 'npm install' command

3. Install mongodb in your machine or you can use free cloud account(https://cloud.mongodb.com/user?n=%2Fv2%2F5bab299297019956e91d0cea#/atlas/login).

4. Go to db.js file in db folder and change your connection string.

5. Run 'node app' command in command prompt.
